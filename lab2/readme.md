## how to start

- install `npm i` and then start server `npm start`
- visit `http://localhost:3000/`
- i hope this works in linux it should work I added "path" convention + \_\_dirname which detects current route and help to serve that index.bootstrap.html file
- if there is any issue please ask me I will happy to fix

## basic api endpoints

```
GET     /items
GET     /items/2
POST    /items        EX. { "name": "Yellow" }
PUT     /items/1      EX. { "name": "Purple" }
DELETE  /items/3
```
